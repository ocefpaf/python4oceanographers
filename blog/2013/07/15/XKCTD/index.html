XKC(T)D plot fun

Jul 15, 2013

Jakecob Vanderplas Sketch Path Filter got into the core of matplotlib 1.4.x. Now it is easy to produce plots with the XKCD style with just the following:

In [2]:
import matplotlib.pyplot as plt
plt.xkcd()  #<- That's it!
Out[2]:
<matplotlib.rc_context at 0x4fef910>

You will need the latest development version of matplotlib. (If you use openSUSE 12.3 just add my custom repository, here is a list of the packages available).

And here is my XKC(T)D profile ;-)

In [12]:
fig, ax = downcast['t090C'].plot()
ax.set_title("XKC(T)D")
ax.set_xlabel("Temperature")
ax.set_ylabel("Pressure [dbar]")
fig.set_size_inches((6, 8))
